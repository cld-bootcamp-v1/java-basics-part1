1.Max of Three Numbers
Write a program that takes three numbers as input and prints the largest using if-else.

2.Grade Calculator
Write a program that takes a student’s score (0-100) and prints their grade based on this scale:
A: 90-100, B: 80-89, C: 70-79, D: 60-69, F: 0-59.

3.Check Positive, Negative, or Zero
Write a program that checks if an integer is positive, negative, or zero using if-else.

4.Days of the Week (Switch)
Write a program that takes an integer (1-7) as input and prints the corresponding day of the week (1 = Monday, 7 = Sunday) using a switch statement.

5.Even or Odd (Ternary Operator)
Write a program that takes a number and prints “Even” if it’s even and “Odd” if it’s odd using the ternary operator.

6.Bank Account Type (Switch)
Write a program that takes a bank account type ("Savings", "Current", "Fixed Deposit") as input and prints the interest rate for each type using a switch statement.

7.Eligibility to Vote (If-Else)
Write a program that checks if a person is eligible to vote. They must be at least 18 years old.

8.Find Largest of Two Numbers (Ternary Operator)
Write a program that takes two numbers as input and prints the largest number using the ternary operator.

9.FizzBuzz
Write a program that prints the numbers from 1 to 100. For multiples of 3, print “Fizz”, for multiples of 5, print “Buzz”, and for multiples of both 3 and 5, print “FizzBuzz”.

10.Traffic Light System (Switch)
Write a program that simulates a traffic light system. Based on the input color ("Red", "Yellow", "Green"), print the corresponding action ("Stop", "Caution", "Go").
